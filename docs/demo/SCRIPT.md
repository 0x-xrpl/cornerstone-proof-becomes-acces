# 30-Second Script

Use this exact structure if you want the demo to stay concise.

## Open
CornerStone turns support into access.

## Step 1
I connect a wallet, but the wallet is not the story.

## Step 2
I complete `Watch Check-in` and `Quiz Pass`.
These become recognized support proof.

## Step 3
Because the required support was recognized, the `Corner AMA Key` unlocks.

## Step 4
That key is not a badge. It opens a real access route right now.

## Optional close
Proof is recognized support. The key is the access right.
